git branch: ver los usuarios

git branch user2: crea una rama

GIT HUB agregar el repositorio de github

git remote add origin url

git push -u origin master o el nombre de la rama que se desea

GIT HUB eliminar el repositorio de github

git remote rm origin

git remote -mv

GIT HUB clonar el repositorio de github

git clone url

git merge branch(hernandez) fusionar o enviar a la rama master

git branch -d hernandez ==> elimina la rama
